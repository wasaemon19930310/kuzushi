# 崩し字解析 Webアプリケーション
AI（ディープラーニング）を用いて、日本古典籍の崩字を画像から解読するWebアプリです。

## 概要
古文書などに使われる崩字をデジタルカメラやスマホで撮影しアップロードすると、AIが解析して対応するUnicode文字とその文字の精度を表示します。

## 技術スタック
- **Backend**: Python 3.x, Flask
- **AI/ML**: Keras, TensorFlow
- **Dataset**: 日本古典籍くずし字データセット（人文学オープンデータ共同利用センター）
- **Environment**: Apache + mod_wsgi (VPS)

## こだわり・工夫した点
- **高精度な解析**: CNN（畳み込みニューラルネットワーク）を採用し、多クラス分類を実現しました。
- **UI/UX**: 解析結果から「文字形状データベース」へワンクリックで飛べるようにリンクを動的生成し、翻刻の補助ツールとしての利便性を高めました。
- **実務への応用**: 以前、実際にVPS上で一般公開し、運用を経験しました。
